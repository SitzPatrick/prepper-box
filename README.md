# prepper-box

Sanitized infrastructure repo for Patrick's Ubuntu prepper box.

Included here:
- Homepage configuration
- Audiobookshelf configuration
- Samba share configuration for Books and Audiobooks
- Network admin app code
- Paperless and Qdrant compose files

Excluded from git:
- secrets, tokens, Wi-Fi passwords
- media files and downloaded content
- databases, cache files, logs, and generated artifacts

This repo is a cleaned, shareable subset of the box configuration rather than a full disk image.
